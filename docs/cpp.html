class DimensionBase¶
#include <DimensionBase.hpp>

Abstract base class for describing dimenions The general idea behind the Dimension class is that it is used to describe the dimension of an array/vector Derived classes are expected to provide the following functionality:

  1. Encapsulate functions (notably index functions) which are specific to the dimension
  2. Allow serialization to read from/write to archives and other processes
  3. Keep track of permutations so that an underlying linked array may be permuted but the indexing stays the same
  4. Provide ways to compare and permute other Dimensions so that they can be sorted relative to each other
  5. Can be used as template arguments in order to select the correct functions at compile time (e.g. an observation equation) Please see the class TimeDimension for an example of a derived class This abstract base class makes sure that the pure virtual functions are implemented and that containers can store (smart) pointers to DimensionBase.

Subclassed by frommle::TimeDimension

class TimeDimension : public frommle::DimensionBase¶
#include <TimeDimension.hpp>

Holds a vector of time stamps (possibly irregular) The TimeDimension is essentially a vector with time stamps. Internally it stores a vector of std::chrono::time_point This can then be converted to different kinds of time formats.
